Node.js

. What is Node.js?

Node.js® is a JavaScript runtime built on Chrome’s V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js’ package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

- https://nodejs.org

Node.js는 오픈소스 서버 프레임워크로 Windows, Linux를 비롯한 다양한 플랫폼에서 동작한다.


. Why Node.js?

Node.js uses asynchronous programming!
A common task for a web server can be to open a file on the server and return the content to the client.

Here is how PHP or ASP handles a file request:
(1) Sends the task to the computer’s file system.
(2) Waits while the file system opens and reads the file.
(3) Returns the content to the client.
(4) Ready to handle the next request.

Here is how Node.js handles a file request:
(1) Sends the task to the computer’s file system.
(2) Ready to handle the next request.
(3) When the file system has opened and read the file, the server returns the content to the client.

Node.js eliminates the waiting, and simply continues with the next request.
Node.js runs single-threaded, non-blocking, asynchronously programming, which is very memory efficient.

예전에 프로젝트할 때는 써야한다고 하니까 썼었는데, 메모리를 정말 효율적으로 사용하는 아이였다. 위에서도 이미 언급되었지만 I/O의 결과를 직접 돌려받는 대신, 콜백 함수의 파라미터로 전달받는다. 다시 말해, 디스크에 결과를 요청하고 기다리는 대신 다른 태스크를 진행하다가 응답 이벤트가 발생하면 태스크의 결과를 처리하는 식인데 이게 정말 엄청난 특징! 왜 채팅 프로그램에서 가장 유용한 아이였는지 이해하게 되었다.

require 함수는 자바스크립트 API의 인터페이스로 해당 모듈의 object를 리턴한다. 여기서 require 안에는 파일의 디렉토리 경로나 파일 이름이 들어갈 수 있으며 Node의 라이브러리가 제공하는 기본 모듈인 HTTP, Crypto, Process, Events, Buffers, File System, Path 등의 모듈은 기본적으로 require 함수로 로드할 수 있다. 기본적인 모듈이 아닌 외부 모듈이라면 NPM(Node Package Manager)으로 다운로드하여 로드할 수 있다.

. What is a Module in Node.js?

Consider modules to be the same as JavaScript libraries.
A set of functions you want to include in your application.

. Built-in Modules

Node.js has a set of built-in modules which you can use without any further installation.

Module Description
buffer To handle binary data
crypto To handle OpenSSL crptographic functions
events To handle events
fs To handle the file system
http To make Node.js act as an HTTP server
https To make Node.js act as an HTTPS server
os Provides information about the operating system
path To handle file paths

NPM(Node.js Package Manager)

. What is NPM?

NPM is a package manager for Node.js packages, or modules if you like.
www.npmjs.com hosts thousands of free packages to download and use.
The NPM program is installed on your computer when you install Node.js

. What is a Package?

A package in Node.js contains all the files you need for a module.
Modules are JavaScript libraries you can include in your project.

모듈이 Node.js의 기본 모듈 API를 사용하여 만든 라이브러리라면 패키지는 모듈(라이브러리)을 사용하기 위한 물리적인 개념으로 이해할 수 있을 것 같다. 검증된 Third-party Package를 쉽게 다운로드하여 사용할 수 있어 생산성이 큰 장점으로 꼽히며, NPM을 통해 설치된 패키지들은 관리가 수월하다.
